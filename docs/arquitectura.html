Stack tecnológico
Apache Airflow
2.9.3
Orquestador del pipeline ETL. Define el orden y la frecuencia de ejecución de cada job.
Apache Spark
3.5 (bitnami)
Motor de procesamiento distribuido. Lee de Postgres, transforma con DataFrames y escribe en Hive.
Apache Hive
3.x
Data Warehouse. Almacena el esquema estrella (dimensiones, hechos y vistas OLAP).
Metabase
v0.61
Herramienta de BI. Conecta a HiveServer2 vía SparkSQL para visualizar los dashboards OLAP.
Neo4J
5.20
Base de datos de grafos. Modela co-compras, usuarios influyentes y rutas de entrega.
PostgreSQL
16-alpine
Base interna del stack. Aloja el metastore de Hive, la metadata de Airflow y de Metabase.
